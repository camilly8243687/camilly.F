💖 Placa Mãe 💖

1. Processador (CPU)

Responsável por executar as tarefas do computador.

2. Memória RAM

Armazena dados temporários para uso rápido.

3. Chipset

Controla a comunicação entre os componentes.

4. BIOS

Inicializa o sistema ao ligar o computador.

5. Slots PCI

Permitem conectar placas de expansão.

6. Conector ATX

Fornece energia para a placa mãe.

7. HD/SSD

Armazena os dados permanentemente.

8. Portas USB

Permitem conectar dispositivos externos.

9. Placa de Vídeo

Responsável pelos gráficos.

10. Bateria CMOS

Guarda configurações do sistema.
